# 🖩 Console Calculator

Простой и удобный консольный калькулятор на Java, написанный для учебных целей.  
Поддерживает базовые арифметические операции и несколько дополнительных функций.

## ✨ Возможности

- **Бинарные операции**: `+`, `-`, `*`, `/`, `%`, `^` (возведение в степень)
- **Унарные операции**: `sqrt` (квадратный корень)
- Обработка ошибок (деление на ноль, отрицательный корень, неверный ввод)
- Формат вывода с точностью до 4 знаков после запятой
